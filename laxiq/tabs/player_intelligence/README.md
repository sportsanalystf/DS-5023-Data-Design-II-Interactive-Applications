# Player Intelligence Tabs

Each file handles one tab on the Player Intelligence page:

- `team_overview.py` - roster table with tier rankings and charts
- `player_cards.py` - individual player profiles with radar charts
- `player_comparison.py` - head-to-head comparison tool
- `draw_control.py` - draw control analytics and scouting clips
- `goal_tending.py` - goalkeeper stats and analysis
- `metrics.py` - scoring/impact calculations shared across tabs
- `charts.py` - plotly chart builders shared across tabs
